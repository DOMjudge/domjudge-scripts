Debian Packages

There are also Debian GNU/Linux packages available from http://non-gnu.uvt.nl/debian/squeeze/domjudge/ for i386 and amd64 architectures. The packages work on Lenny, Squeeze and newer, and also on the various versions of Ubuntu.

You can add the following apt source to /etc/apt/sources.list which is signed with this key:

deb http://non-gnu.uvt.nl/debian squeeze uvt

DOMjudge-live images

We also provide DOMjudge-live images to run DOMjudge straight from an USB stick (it can also be written to harddisk or booted in a virtual machine). See this README file (which is also available in the DOMjudge packaging Git repository and as /README on the image) more details on how use these.
